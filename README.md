## 如何测试

1. 安装 `php` 和 `composer` 

- PHP：https://www.php.net/manual/zh/install.php
- Composer：https://www.phpcomposer.com

2. 在测试目录，安装 `vendor` 依赖

```bash
composer install
```

3. 测试代码

```shell
php index.php
```
